#How to start the project

 
 1.     docker build -t <DOCKERHUB>/laravel-demo
 2.     docker run --rm -t -p 9000:80 <DOCKERHUB>/laravel-demo (do not change the port)    
 3.     <p>optional: If you wish to push it to the docker registry execute this command: </p>docker push <DOCKERHUB>/laravel-demo
